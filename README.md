# TreidPro

Динамический интернет-магазин на Django с администраторской частью и пользовательской витриной.

## Возможности
- Редактируемое меню: каталог товаров, доставка, оплата, новости, акции, о нас, корзина.
- Каталог товаров с категориями, поиском и карточками товаров.
- Главная страница с большим banner в header, каруселью, секцией «Вы просматривали» и блоком «Рекомендуем».
- Разделы новостей, акций и информационных страниц.
- Простая корзина на сессиях.
- Полноценная админ-панель Django для управления контентом.

## Запуск
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```
